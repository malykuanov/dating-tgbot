# Телеграм-бот для знакомств

Сервис для знакомств в Телеграм

Используется:
- Python 3.8
- pyTelegramBotAPI
- Django 4.0
- PostgreSQL

В боте реализовано:
- регистрация профиля пользователя
- выбор параметров для поиска профилей пользователей
- поиск пользователей в соответствии с настройками
